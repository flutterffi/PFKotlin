# Playground

This folder is your free practice area.

Use it when you want to:

- copy code from lessons and modify it freely
- try ideas before moving them into the real project
- keep temporary experiments separate from the main learning path

## Subfolders

- `scratch/`: throwaway experiments
- `kata/`: tiny self-made drills
- `sandbox_data/`: temporary input files

You can create as many small files here as you want while studying.
