# 04 Learning Hub Store

This project is an advanced Kotlin app architecture practice.

## Focus

- offline-first bootstrap and refresh flows
- repository, local store, and remote source boundaries
- reducer-driven app state
- route state for dashboard and detail screens
- one-shot notices and failure states
- controlled async effects for testing

## Commands

```bash
kotlinc projects/04_learning_hub_store/*.kt -include-runtime -d learning-hub.jar
java -jar learning-hub.jar
```
