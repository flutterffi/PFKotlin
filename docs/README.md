# Docs

This folder collects study navigation and repository guides.

## Files

- `kotlin_official_study_map.md`: official Kotlin links mapped to local practice files
- `kotlin_7_day_bootcamp.md`: a day-by-day Kotlin sprint plan using this repository
- `practice_workspace_guide.md`: where to read, practice, and experiment

## Suggested use

- open one official page
- open the matching local practice file
- make one change
- rerun the example or test
