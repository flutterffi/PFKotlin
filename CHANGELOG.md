# Changelog

## 0.1.0 - 2026-05-28

- Created the initial `PFKotlin` practice repository structure.
- Added foundation lessons for collections, null safety, and extension functions.
- Added advanced lessons for sealed types, sequences, and `Result`.
- Implemented a study planner CLI mini project for Kotlin syntax and design practice.

## 0.2.0 - 2026-05-28

- Added command-line argument parsing for the study planner CLI.
- Added file-based task loading with validation and readable error messages.
- Added a sample study task dataset for repeatable Kotlin parsing practice.

## 0.3.0 - 2026-05-28

- Added report export support to the study planner CLI with a new `--save` flag.
- Added a new advanced lesson for sealed parser outcomes and explicit error modeling.
- Expanded the README examples to cover file export and parser practice.
